JustTalk

JustTalk is a chat application with only one chat in which you can talk about anything and everything. This project was developed entirely by me in HTML/CSS/JS for the frontend and in NodeJS with MongoDB for the backend. The goal of this project was to train myself to make a complete backend. PS: If you go to the site, the first loading may be long due to the DB.
